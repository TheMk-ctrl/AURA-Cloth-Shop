# AURA — Концептуальный магазин

Проект представляет собой концептуальный интернет-магазин, ориентированный на минималистичный дизайн, премиальную подачу товаров и бесшовный пользовательский опыт. Основной упор сделан на интерактивные элементы, адаптивность и производительность.

## 🚀 Основные возможности

*   **Интерактивный лукбук**: 3D-анимация коллекций, использующая возможности библиотеки GSAP для создания эффекта глубины и погружения.
*   **Адаптивная архитектура**: Полная мобильная оптимизация: на десктопах реализован сложный 3D-эффект, на мобильных устройствах — плавный горизонтальный скролл.
*   **Система быстрого просмотра (Quick View)**: Позволяет изучить детали товара, выбрать цвет и размер, не покидая текущую страницу.
*   **Корзина с «умным» прогресс-баром**: Визуализация условий для бесплатной доставки в режиме реального времени.
*   **Фильтрация и поиск**: Динамический каталог с возможностью сортировки по цене и фильтрации по категориям.
*   **Система уведомлений (Toast)**: Интерактивное оповещение пользователя о добавлении товаров в избранное или корзину.

## 🛠 Технологический стек

*   **Frontend**: HTML5, CSS3 (Tailwind CSS для быстрой и масштабируемой верстки).
*   **Анимации**: GSAP (GreenSock Animation Platform) + ScrollTrigger для сложной логики скролла.
*   **Иконки**: Lucide Icons.
*   **Логика**: Vanilla JavaScript (ES6+), работа с `localStorage` для сохранения состояния корзины и списка избранного.

## 💡 Решение ключевых задач

1.  **Производительность**: Отказ от тяжелых кастомных курсоров в пользу нативных взаимодействий, что значительно снизило нагрузку на CPU на мобильных устройствах.
2.  **UX-дизайн**: Двухшаговая система оформления заказа (через drawer) позволяет пользователю завершить покупку, оставаясь в контексте сайта, не переходя на страницу оплаты.
3.  **Адаптивность**: Реализована логика, которая отключает "тяжелые" 3D-анимации на экранах смартфонов, заменяя их на удобный `touch-friendly` интерфейс.

## 📈 Планы по развитию
*   Интеграция с реальным API (например, Stripe для платежей).
*   Перенос логики на React/Next.js для улучшения серверного рендеринга и управления состоянием через Redux/Zustand.
*   Добавление системы рекомендаций товаров на базе интересов пользователя.
